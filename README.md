# Weekend Bot

Discord bot for partying on the weekend. Keeps track of everyone's drink count with -cheers


## Tech Stack

**NodeJs** - v16.14.0\
**npm** - v8.5.4

https://nodejs.org/dist/v16.14.2/node-v16.14.2-x64.msi


## Deployment

To deploy this project run

```bash
  npm i
```
then
```bash
  node .
```
